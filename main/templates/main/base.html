    Bem vindo usuário!


    Ao Software de cadastro e consulta de clientes desenvolvido sob iniciativa da disciplina Projeto Integrador III da Universidade Virtual do Estado de São Paulo (UNIVESP), envolvendo os cursos de Ciência de Dados e Engenharia da Computação.


    Orientações de uso:

    --> Para Cadastrar um novo cliente ou consultar os dados de um cliente já cadastrado clique em "+ Cadastrar" no menu superior do seu navegador.


    --> Para editar os dados de um cliente já cadastrado clique no botão "Editar" da seção "Ações" na página de Cadastro de Clientes.


    --> Para excluir o registro de um cliente já cadastrado clique no botão "Excluir" da seção "Ações" na página de Cadastro e Consulta de Clientes.


    --> Para consultar os dados de algum cliente já cadastrado utilize a caixa de busca por nome e clique no botão "Buscar" para realizar a pesquisa.
